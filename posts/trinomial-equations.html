Триноміальні рівняння

2025-02-11

Що таке триноміальні рівняння

Триноміальні рівняння — це особливий тип поліноміального рівняння, що складається з трьох доданків, які містять стали та степені змінної.

\[ ax^{2n} + bx^{n} + c = 0 \]

Така форма особливо корисна, оскільки вона дозволяє використовувати стратегії заміни, наприклад, встановити \(x^n = y\), щоб перетворити рівняння на стандартне квадратне.

  • \( a \), \( b \), та \( c \) — це числові коефіцієнти.
  • \( x \) — це невідома змінна.

Варто зауважити, що якщо \( n = 1 \), рівняння стає квадратним рівнянням, яке часто записують у вигляді:

\[ ax^2 + bx + c = 0 \]

Як розв'язувати триноміальні рівняння

Для розв'язання триноміального рівняння вигляду \( ax^{2n} + bx^n + c = 0 \) можна використовувати загальний набір кроків. Першим кроком є виконання заміни, яка перетворює рівняння на квадратне рівняння відносно \( y \), яке легше розв'язати. Наприклад, нехай:

\[ x^n = y \]

і отримаємо:

\[ ay^2 + by + c = 0 \]

Ми можемо використовувати такі методи, як розкладання на множники, доповнення до повного квадрата або формулу коренів квадратного рівняння, щоб розв'язати отримане квадратне рівняння відносно \( y \). Після того як ми знайдемо розв'язання для \( y \), ми повертаємося до оригінальної заміни \( y = x^n \), щоб визначити відповідні значення \( x \). Коротко:

  • Нехай \( y = x^n \), тоді рівняння набуває вигляду \( ay^2 + by + c = 0 \).

  • Розв'яжіть квадратне рівняння відносно \( y \).

  • Для кожного розв'язання \( y_i \) розв'яжіть \( x^n = y_i \), щоб знайти відповідні значення \( x \).

  • Перевірте всі розв'язання в початковому рівнянні, щоб переконатися в їхній правильності.

Важливо перевірити кожне розв'язання, підставивши його в початкове рівняння, щоб підтвердити його правильність. Цей процес може здатися складним, але насправді він практичний і простий у виконанні.

При розв'язуванні рівнянь за допомогою заміни іноді можуть з'являтися додаткові розв'язання, які називають сторонніми розв'язаннями. Вони можуть задовольняти перетворене рівняння, але не задовольняти початкове. Підставляючи кожне розв'язання назад у початкове рівняння, ми переконуємося, що воно дійсно працює і не порушує жодних неявних умов.

Приклад

Розв'яжіть рівняння \(3x^4-7x^2 + 2 = 0\).

Нехай \(y = x^2\), щоб перетворити рівняння на квадратне рівняння:

\[3y^2-7y + 2 = 0\]

Ми можемо використати формулу коренів квадратного рівняння, щоб знайти значення \(y\). Отримаємо:

\begin{align*} y &= \frac{{-(-7) \pm \sqrt{{(-7)^2-4 \cdot 3 \cdot 2}}}}{{2 \cdot 3}} \\[0.6em] &= \frac{{7 \pm \sqrt{{49-24}}}}{6} \\[0.6em] &= \frac{{7 \pm \sqrt{25}}}{6} \\[0.6em] &= \frac{{7 \pm 5}}{6} \end{align*}

Отже, ми знайшли:

\[y_1 = \frac{12}{6} = 2\] \[y_2 = \frac{2}{6} = \frac{1}{3}\]

Після того як розв'язання для \(y\) знайдено, їх можна підставити назад у початкове рівняння за допомогою \(x^2 = y\), щоб знайти відповідні значення \(x\).

Для \(y = 2\) маємо: \[x^2 = 2 \quad \rightarrow \quad x = \pm \sqrt{2} \]

Для \(y = \frac{1}{3}\) маємо \[x^2 = \frac{1}{3} \quad \rightarrow \quad \pm \sqrt{\frac{1}{3}} \]

Необхідно перевірити правильність розв'язань, підставивши їх назад у початкове рівняння для забезпечення точності. Щоб перевірити, чи є задані значення розв'язаннями рівняння \(3x^4-7x^2 + 2 = 0\), ми повинні підставити їх у рівняння та перевірити, чи задовольняють вони його.

Для \( x = \sqrt{2} \):

\[ 3(\sqrt{2})^4-7(\sqrt{2})^2 + 2 = 3 \cdot 2^2-7 \cdot 2 + 2 \] \[ = 12 – 14 + 2 = 0 \]

Для \( x = -\sqrt{2} \): \[ 3(-\sqrt{2})^4-7(-\sqrt{2})^2 + 2 = 3 \cdot 2^2-7 \cdot 2 + 2 \] \[ = 12 - 14 + 2 = 0 \]

Для \( x = \sqrt{\dfrac{1}{3}} \): \[ 3\left(\sqrt{\frac{1}{3}}\right)^4-7\left(\sqrt{\frac{1}{3}}\right)^2 + 2 = 3 \cdot \left(\frac{1}{3}\right)^2 - 7 \cdot \frac{1}{3} + 2 \] \[ = \frac{3}{9}-\frac{7}{3} + 2 = 0 \]

Для \( x = -\sqrt{\dfrac{1}{3}} \): \[ 3\left(-\sqrt{\frac{1}{3}}\right)^4-7\left(-\sqrt{\frac{1}{3}}\right)^2 + 2 = 3 \cdot \left(\frac{1}{3}\right)^2-7 \cdot \frac{1}{3} + 2 \] \[ = \frac{3}{9}- \frac{7}{3} + 2 = 0 \]

Усі розв'язання задовольняють початкове рівняння.

Розв'язанням рівняння є: \[x = \pm \sqrt{2} \quad \text{та} \quad \pm \sqrt{\frac{1}{3}}\]
