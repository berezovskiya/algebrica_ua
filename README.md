# algebrica_ua

🌐 **Живий сайт:** https://berezovskiya.github.io/algebrica_ua

Український переклад сайту [algebrica.org](https://algebrica.org) — відкрита математична енциклопедія.

## Про проект

213+ статей з математики українською мовою:
- Комплексні числа
- Похідні та інтеграли
- Тригонометрія
- Рівняння
- Теорія ймовірностей та статистика
- Лінійна алгебра
- Ряди та послідовності

## Для контриб'юторів

Всі охочі можуть виправляти помилки перекладу або покращувати статті.

### Як внести зміни

1. **Fork** репозиторію
2. Створіть гілку: `git checkout -b fix/назва-статті`
3. Виправте статтю у відповідному файлі `posts/*.html`
4. Створіть **Pull Request**

Дивіться повну інструкцію в [CONTRIBUTING.md](CONTRIBUTING.md).

## Структура

```
posts/          — статті (HTML)
images/         — зображення
graph/          — інтерактивні графіки
mathjax/        — рендеринг математичних формул
```
