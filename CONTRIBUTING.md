# Внесення змін

## Як подати виправлення

1. **Fork** цього репозиторію
2. Створіть гілку для ваших змін: `git checkout -b fix/назва-статті`
3. Внесіть виправлення у відповідному HTML файлі в папці `posts/`
4. **Перегляньте зміни локально** перед PR
5. Створіть **Pull Request** з описом що виправили

## Правила оформлення

- Виправляйте лише переклад, не змінюйте структуру HTML
- Перевіряйте математичні формули (LaTeX)
- Якщо знайшли помилку в оригіналі (algebrica.org) — це теж цінно, вкажіть в PR

## Структура файлів

```
posts/          — статті (один файл = одна стаття)
images/         — зображення для статей
graph/          — інтерактивні графіки
```

## Перевірка

Перед PR переконайтесь, що:
- [ ] HTML валідний
- [ ] Формули відображаються коректно
- [ ] Зображення завантажуються
